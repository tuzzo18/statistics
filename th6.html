Th6: Algorithms for random variates generation

In probability and statistics, a random variate or simply variate is a particular outcome of a random variable; the random variates which are other outcomes of the same random variable might have different values (random numbers).

Generating random variates is a fundamental aspect of many computational simulations and statistical methods. There are various algorithms for generating random variates for different probability distributions. The choice of method depends on the specific distribution you are working with. Some methods may be more efficient or easier to implement for certain distributions.

Here's a brief overview of some commonly used methods:

Unfirom Distribution

  • Inverse Transform Sampling: If \(U\) is a random variable uniformly distributed between 0 and 1, and \(F^{-1}\) is the inverse of the cumulative distribution function (CDF) of the desired distribution, then \(F^{-1}(U)\) follows the desired distribution.
  • Linear Congruential Generators (LCGs): These are simple algorithms that generate pseudo-random numbers using a linear recurrence relation. While they are not suitable for all applications due to their periodic behavior, they are often used for generating uniform random variates.

Normal Distribution

  • Box-Muller Transform: Generates two independent standard normal random variables from two independent uniform random variables.
  • Marsaglia Polar Method: Another method for generating standard normal variates, using polar coordinates.

Exponential Distribution

  • Inverse Transform Sampling: Similar to the uniform distribution, if \(U\) is uniformly distributed and \(F^{-1}\) is the inverse of the exponential CDF, then \(F^{-1}(U)\) follows an exponential distribution.
  • Ratio-of-Uniforms Method: Involves generating random variates from a double-exponential distribution and then transforming them to follow an exponential distribution.

Poisson Distribution

  • Knuth's Algorithm: Uses a recursive formula based on uniform random variables.
  • Inversion by Sequential Search: Involves generating a sequence of exponential random variables until the sum exceeds a threshold.
These are just a few examples, and there are many other specialized algorithms for different distributions. Additionally, with the advent of computer software and libraries, many programming languages provide built-in functions for generating random variates from different distributions. These functions often use more advanced algorithms and techniques for efficiency and accuracy.
